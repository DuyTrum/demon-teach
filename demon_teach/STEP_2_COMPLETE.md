## How to Test Crashlytics

### Option 1: Add Test Crash Button (Debug Only)

Add this to your test menu or settings screen:

```dart
import 'package:flutter/foundation.dart';
import 'package:firebase_crashlytics/firebase_crashlytics.dart';

// In your widget
if (kDebugMode) {
  ElevatedButton(
    onPressed: () {
      // Force a crash to test Crashlytics
      FirebaseCrashlytics.instance.crash();
    },
    child: Text('Test Crash (Debug Only)'),
  );
}
```

### Option 2: Throw Test Exception

```dart
ElevatedButton(
  onPressed: () {
    throw Exception('Test crash for Crashlytics');
  },
  child: Text('Test Exception'),
);
```

### Verify in Firebase Console

1. Go to https://console.firebase.google.com
2. Select project: **demon-teach**
3. Click **Crashlytics** in left menu
4. Wait 5-10 minutes for crash to appear
5. Refresh page to see crash report
